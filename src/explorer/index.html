About polkax402 Explorer

This is a lightweight block explorer for the polkax402 blockchain network. Built with a classic Mac OS aesthetic to celebrate the golden era of computing.

Features:

  • Search blocks by number or hash
  • Search transactions by hash (extrinsic hash)
  • Search accounts by address
  • View real-time blockchain data
  • Track HTTPUSD token balances
  • Monitor network status
  • Retro 90's Apple interface

Search Examples:
• Block number: 12345
• Block hash: 0x1234...
• Transaction hash: 0xabcd...
• Account address: 5GrwvaEF...
